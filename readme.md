# Решения лабораторных работ по дисциплине "Программирование для ОС"

В следующем списке указано соответствие файлов с исходным кодом программ решаемым задачам.

1. culculus. Вы делаете программу для автобусного сообщения. Автобус может перевозить одновременно 50 пассажиров. Имея в качестве входных данных количество пассажиров, ожидающих на автовокзале, нужно рассчитать и вывести, сколько свободных мест будет в последнем автобусе.

2. countdown. Вам нужно сделать приложение обратного отсчета. На входе задано число N, выведите числа от N до 1 в отдельных строках. Кроме того, когда текущее число обратного отсчета кратно 5, приложение должно выводить «Beep».

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Tweedledum-second/culculus.git
cd culculus/src
```

2. Скомпилируйте нужный файл, например, culculus.cpp:

```bash
g++ culculus.cpp -o culculus
```

3. Запустите исполняемый файл culculus:

```bash
./culculus
```
