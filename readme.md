# Программа для нахождения разности делителя и остатка от целочисленного деления некоторого натурального числа

Программа предназначена для решения задачи такой формулировки: Вы делаете программу для автобусного сообщения. Автобус может перевозить одновременно 50 пассажиров. Имея в качестве входных данных количество пассажиров, ожидающих на автовокзале, нужно рассчитать и вывести, сколько свободных мест будет в последнем автобусе.

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Tweedledum-second/culculus.git
cd culculus/src
```

2. Скомпилируйте файл culculus.cpp:

```bash
g++ culculus.cpp -o culculus
```

3. Запустите исполняемый файл culculus:

```bash
./culculus
```


## Использование

### Пример

```bash
Пожалуйста, введите количество пассажиров на остановке: 432
Осталось свободных мест: 18
```
